# IND8 Tracker – Indirect Work Tracking System

IND8 Tracker is a lightweight desktop application designed to track Direct and Indirect labor time for associates. It includes automatic session handling, violation warnings, dashboards, CSV exports, and a clean Amazon‑style UI.

---

## Features

- Badge scanning and auto‑associate creation  
- Direct and Indirect session tracking  
- Automatic session closing when switching tasks  
- 5‑hour warning and 6‑hour violation alerts  
- Dashboard for all associates  
- CSV export and shift reports  
- Custom icon and splash screen  
- Portable mode (no installation required)  
- Optional cloud‑sync mode for using the app at home and work
